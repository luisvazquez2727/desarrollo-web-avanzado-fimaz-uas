# Práctica de Laboratorio 1: Creación de Clases y Encapsulamiento en PHP

## Objetivo
Aplicar los fundamentos de la Programación Orientada a Objetos en PHP, implementando una clase con atributos privados, constructor y métodos de acceso, siguiendo buenas prácticas de encapsulamiento.

## Descripción
Se creó la clase `Usuario` con:
- Atributos privados: `nombre` y `correo`
- Constructor para inicializar valores
- Métodos getters (`getNombre`, `getCorreo`)
- Métodos setters (`setNombre`, `setCorreo`)

## Instrucciones de ejecución
1. Ubicar los archivos en la ruta que se allan guardado
2. Abrir `index.php` en un servidor local (ejemplo: XAMPP, o el server local de php).
3. Visualizar en el navegador los valores del usuario mostrados con los métodos getters.
